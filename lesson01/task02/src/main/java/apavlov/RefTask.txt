Задание:

Посмотрите данный код и ответьте на вопросы.
[code=java]
public class RefTask {
public static void main(String[] args) {
Integer value = 1;
RefTask.change(value);
System.out.println(value);
}

public static void change(Integer value) {
value++;
}
}

[/code]

Что будет выведено на консоль и почему?

*********************************************************************

Решение:

Ответ: в консоль будет выведено 1

в методе "main" мы объявляем переменную value типом Integer и присваиваем
ей значение 1

Integer - это оболочка примитивного типа int, но также является полнофункциональным
объектом, что означает, что переменная value имеет лишь ссылку на объект, которая
неявно преобразуется в значение...

Далее, в методе main мы вызываем метод change(Integer value);
Как видно, в этот метод мы передаём ссылку на объект Integer...
Стоит помнить, что объект Integer никогда не меняет своего значения, поэтому
при каждом инкременте создаётся новый объект Integer.
Следовательно, что при value++ после увелечения переменной на +1 на выходе
переменная value будет иметь новую ссылку на объект Integer, которая будет отличаться
от первоначальной ссылки. А т.к. метод не возвращает ссылку на Integer, то в методе
main останется ссылка на первый объект, где значение равно 1
В итоге в консоли мы увидим 1
