Задание:

Можно ли ваше решение из задание 2 адаптировать к поиску максимально значения
для других фигур. Квадрат, Многоугольник?

*************************************************************************************

Ответ:

Возможно...

Я постарался продемонстрировать один из способов...
Есть интерфейс Figure, в котором описано 2 метода:
 double[] longLines(); - возвращает массив длин всех сторон фигуры
 double area();        - возвращает значение площади фигуры

Класс Triangle имплементирован от интерфейса Figure, так же можно создать
класс для квадрата и многоугольника, в которых реализовать требуемые в интерфейсе
методы

Класс MaxSides, который имеет метод max(Figure figure) не привязан к определённому
типу фигуры  поэтому он работает для всех классов, имплементируемых от интерфейса
Figure, будь-то там квадрат, треугольник или многоугольник

Так что для адаптации затратится минимум времени
